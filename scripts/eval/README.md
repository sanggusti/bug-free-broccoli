# Eval

Evaluation harnesses for measuring agent and model performance against FHIR datasets.

## Purpose

- Run automated evaluations of ADK agents on curated FHIR clinical scenarios
- Compare outputs against ground-truth FHIR resources
- Produce structured evaluation reports (JSON / CSV)

## Usage

```bash
python scripts/eval/run_eval.py --config scripts/configs/eval_config.yaml
```
