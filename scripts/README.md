# Scripts

This directory contains utility scripts for the FHIR-ADK project.

## Sub-directories

| Directory  | Purpose |
|------------|---------|
| `eval/`    | Evaluation harnesses — measure agent/model performance against FHIR datasets |
| `runs/`    | Run management — launch, monitor, and record experiment runs |
| `prompt/`  | Prompt templates and prompt-engineering utilities |
| `configs/` | Configuration files and config-loading helpers |
