# Prompt

Prompt templates and prompt-engineering utilities for ADK agents working with FHIR data.

## Purpose

- Store versioned prompt templates used by ADK agents
- Provide helpers for rendering templates with FHIR resource context
- Track prompt iteration history

## Structure

```
prompt/
├── templates/   # Jinja2 / plain-text prompt templates
└── utils/       # Rendering and formatting helpers
```
