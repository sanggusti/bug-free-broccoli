# App

This directory contains the core application code for the FHIR-ADK project.

## Sub-directories

| Directory | Purpose |
|-----------|---------|
| `mcp/`    | Model Context Protocol server and tools for FHIR resource access |

## Overview

The `app` layer exposes FHIR resources through an MCP-compatible interface so that
ADK agents can query and manipulate clinical data in a structured way.
